# 躺平模擬器 (Tangping Simulator)

## 專案概述
一個面向台灣用戶的互動式「躺平模擬器」網頁應用，輸入薪資與存款，計算你能躺平多久。所有經濟數據皆可追溯至台灣政府官方來源。

## 技術棧
- **框架：** React + Vite
- **動畫：** Framer Motion
- **3D：** React Three Fiber (@react-three/fiber) + @react-three/drei
- **樣式：** Tailwind CSS
- **語言：** TypeScript

## 專案結構
```
Tangping/
├── CLAUDE.md              # 專案說明（本檔案）
├── data/
│   └── sources.md         # 所有數據來源與官方連結（可追溯驗證）
├── src/                   # React 應用原始碼
├── public/                # 靜態資源
└── package.json
```

## 核心功能
1. **輸入介面** — 月薪、存款、選擇城市、生活風格
2. **躺平計算** — 考慮通膨（CPI）、租金通膨、定存利率，逐月模擬存款消耗
3. **視覺體驗** — 3D 場景 + Framer Motion 動畫 + 互動特效
4. **結果分享** — 躺平等級、天數、可分享的結果卡片

## 數據原則
- **所有經濟數據必須可追溯至官方來源**
- 數據來源記錄在 `data/sources.md`
- 主要來源：主計總處、勞動部、衛福部、財政部、央行、健保署、勞保局
- 預設參數皆由官方數據推導，不得使用無來源的數字

## 關鍵預設參數
| 參數 | 值 | 來源 |
|------|------|------|
| 通膨率 | 1.7% | 主計總處 114 年 CPI |
| 租金通膨率 | 2.0% | 主計總處房租類 CPI |
| 定存利率 | 1.7% | 台灣銀行牌告 |
| 無工作社保月費 | 2,155 元 | 健保署 + 勞保局 |
| 薪資中位數 | 37,274 元/月 | 主計總處 113 年統計 |

## 目標用戶
台灣 20-40 歲上班族，主要使用繁體中文。

## 開發指引
- UI 文字使用**繁體中文**
- 貨幣單位為**新台幣（TWD / NT$）**
- 所有計算邏輯需附註數據依據
- 保持程式碼簡潔，避免過度工程化
